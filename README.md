# Employee Performance Dashboard (Excel)

## Overview

This project is an Excel-based Employee Performance Dashboard designed to track employee productivity, working hours, break durations, and overall performance. The dashboard provides a centralized view of workforce performance and helps identify productivity trends across employees.

## Tools & Technologies

- Microsoft Excel
- VBA Macros
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Data Validation

## Features

- Automated data processing using VBA Macros
- Employee productivity and efficiency tracking
- Working hours and break time analysis
- Interactive dashboard with visual reports
- Summary view for performance comparison
- KPI monitoring for workforce evaluation

## Key Insights

- Identified employees with the highest productivity levels
- Compared break durations against performance metrics
- Analyzed overall workforce efficiency
- Highlighted performance trends for better decision-making

## What I Learned

- Creating automated workflows using VBA Macros
- Building interactive Excel dashboards
- Transforming raw data into meaningful insights
- Presenting business information through visual reports
